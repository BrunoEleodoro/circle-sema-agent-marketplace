How it works

One marketplace loop, three accountable actors.

Seller

Publishes safe listings

Wallet-authenticated sellers list user-owned data packs, proof services, files, repos, or links.

Gateway

Normalizes trust context

Sema carries provenance, buyer requirements, verification criteria, and review meaning.

Buyer

Searches and pays with x402

Buyer agents compare listings, ask before spending USDC, and unlock delivery after checkout.

Closeout

Tracks delivery and payout

The API records receipts, fulfillment, reviews, reputation updates, and seller payout status.
