# Circle + Sema Marketplace Video

Remotion source for the hackathon showcase video.

```bash
pnpm --filter @agent-stack-ecosystem-kits/marketplace-video dev
pnpm --filter @agent-stack-ecosystem-kits/marketplace-video render
```

The render command writes:

```text
apps/marketplace-video/out/circle-sema-marketplace.mp4
```

The video follows the repository story: Circle Agent Wallets provide identity,
x402 gates paid delivery, Sema adds shared marketplace semantics, and the API
tracks fulfillment, review, and seller payout state.
